# Stock playbook summary

## 5 gates

1. No-BS Rule
   - revenue actually growing
   - actually profitable on free cash flow
   - more cash than debt
   - pass at least 2 of 3

2. Rule of 40
   - growth + FCF margin

3. Moat Test
   - what is hard to copy
   - whether customers stay if features get copied
   - operating evidence of durability

4. CEO Filter
   - track record
   - alignment / skin in the game
   - founder or long-tenure operator quality
   - delivery evidence

5. Valuation
   - bull / base / bear
   - bear case still has to be attractive
   - do not pay for perfection

## Required extra layers

A real review also covers:
- position sizing
- portfolio role
- ownership plan
- kill-switches
- hold / add / trim / exit behavior
- current stock price
- bear / base / bull intrinsic-value stock-price cases

## Ownership rules

- hold when still excellent and sized reasonably
- trim for portfolio risk, not because a great company went up
- sell when thesis breaks
- separate trimming decisions from thesis decisions

## DCA rule

- if no current holding is 10% or more below its 52-week high, deploy only 50% of new monthly capital
- keep 50% as dry powder
- deploy harder only when a holding is down enough and thesis is intact
- do not average down into broken theses
