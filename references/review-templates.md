# Review templates

## Stock review sections

Use this structure:
- what the company does
- Gate 1
- Gate 2
- Gate 3
- Gate 4
- Gate 5
- intrinsic-value stock-price cases
- final decision
- position sizing
- ownership plan
- kill-switches
- five-principles classification
- bad reasons to sell check
- bad-news triage
- binary sell protocol

## Portfolio review sections

Use this structure:
- investor profile
- portfolio structure
- current positions
- worth-owning check
- portfolio-fit check
- ownership-plan check
- capital allocation plan
- portfolio mistakes to fix
- final actions
- quarterly review cadence

## Tone

- blunt
- specific
- no fake certainty
- no “great company therefore buy” laziness
